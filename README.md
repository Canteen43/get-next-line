# Push-swap
## Description
get_next_line() is a C function that reads from the provided file descriptor and returns the next line until there are no next lines left.
## Requierements
- C Compiler
## Installation
- Clone the repository (e.g. `git clone https://github.com/Canteen43/get-next-line`)
- Run `cc src/testmain.c src/get_next_line.c src/get_next_line_utils.c` to get a testable executable `a.out`
## Usage example
- `./a.out test.txt`
